# 夜间光源工具 (Night Light Tool)

一个简单而实用的网页版夜间光源工具，可以调节光源的色温和亮度，帮助用户在夜间获得舒适的照明环境。

## 功能特点

- 可调节的色温范围：2000K（暖色）到 8000K（冷色）
- 可调节的亮度范围：0% - 100%
- 支持自定义颜色选择
- 两种光源模式：色温模式和自定义颜色模式
- 实时预览光源效果
- 响应式设计，适配各种屏幕尺寸
- 简洁直观的用户界面

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
